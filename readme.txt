== Description ==
Royal Login Architect is a unique PHP-only WordPress utility designed to give your login page a royal style makeover. This plugin focuses on simplicity and efficiency, allowing administrators to easily customize the appearance and messaging of the WordPress login screen without touching any code.

**Key Features:**
*   **Customizable Login Logo:** Choose between an image (SVG) or text-based logo. You can upload a custom Base64 encoded SVG or use the elegant default crown icon provided.
*   **Branded Login Text:** Set custom text for your logo and control its color when using a text-based logo, ensuring brand consistency.
*   **Color Scheme Control:** Adjust the background color of the entire login page, the login form itself, and the primary login button (including dynamic hover states) and its text color.
*   **Custom Login Message:** Display a personalized message above the login form to convey important information or greetings, with support for basic HTML tags.
*   **Simplicity First:** All settings are managed through a straightforward and intuitive interface within the WordPress admin dashboard, leveraging the native Settings API.

Elevate your WordPress login experience with a touch of elegance and consistent branding that truly reflects your site's identity.

This plugin is open source. Report bugs at: https://github.com/ogichanchan/royal-login-architect

== Installation ==
1. Upload to /wp-content/plugins/
2. Activate
